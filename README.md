# Large Language Models to Improve the Understandability of Clinical Guidelines: an Evaluation of Readability Improvements and Unintended Content Changes Produced by GPT-4

This Repository contains code and data for the paper: Large Language Models to Improve the Understandability of Clinical Guidelines: an Evaluation of Readability Improvements and Unintended Content Changes Produced by GPT-4

## File Descriptions

*Final content analysis - raw data.csv: actual data
*Final content analysis - key file.csv: key that explains the data
*project_pipeline.ipynb: Jupyter Notebook containing code used to run the experiments in the paper
